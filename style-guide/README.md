# NTR Style Guide — Folder Index

Use this folder to build any new page or microsite that matches the Not The Rug visual identity.

## Files

| File | Purpose |
|------|---------|
| `STYLE-GUIDE.md` | Full design system reference — colors, fonts, spacing, components, do/don't rules |
| `boilerplate.html` | Ready-to-use HTML starter with all CSS tokens, components, and example sections wired up |
| `color-swatches.html` | Open in browser for a visual color reference with hex values and CSS variable names |

## Logo Assets (in `/logos/`)

| File | Use |
|------|-----|
| `ntr_offwhite_horiz.png` | Nav logo on dark (sage-dark) backgrounds |
| `ntr_logo_green_horiz.png` | Logo on light (cream/white) backgrounds |
| `logo_horiz.png` | Standard horizontal lockup |
| `circle_logo.jpg` | Avatar / social / profile use |
| `Not the rug logo design.png` | Full design reference |
| `Not_The_Rug_2023_clipped_web.mp4` / `.webm` | Hero background video (looping, muted) |

## Quick Start

1. Copy `boilerplate.html` to your new project
2. Update the logo `src` path relative to your new file location
3. Reference `STYLE-GUIDE.md` for any component or token you need
4. Open `color-swatches.html` in a browser to visually confirm colors
